# flutter_html_editor

A package project of Flutter HTML Editor based on summernote.

## Getting Started

Add package to your project

```dart
dependencies:
  flutter_html_editor:
    git:
    url: ../../opensource/flutter_html_editor/flutter_html_editor
```
